# 开源对比评分

``` 
评分标准
DALL-E: DALL-E 绘画支持

MP: 微信公众号对话支持
VDB: 向量数据库支持，上传文件对话支持
Member: 会员系统，收费系统
Beauty: UI美观度
Status: 最近更新时间 2023-06-30 ， 距离当前时间每多1天多扣0.1
```
